# About

Welcome to the Void Mod, A small mod that adds a new advancement tree called The Void.
To begin your journey you will need to seach end city chests for an Enchantment called Soul Reaper.
Add Soul Reaper to your Sword.
Begin killing Endermen with Soul Reaper on your sword, you have a 1/100 chance that an item called Void Soul will drop.
Use 4 Void Soul & a Compass to craft The Void Compass.
Link the Void Compass to a lodestone, when you next interact with the Void Compass you will teleport back to the Lodestone, No matter where you are or what Dimension you are in!

## Setup

Version: Minecraft Fabric 26.1

Fabirc API is required
